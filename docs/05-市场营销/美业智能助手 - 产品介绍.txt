# 美业智能助手 - 产品介绍

## 产品概述

美业智能助手是一款面向美容美发门店的**对话式管理助手 Web 应用**，采用类 ChatGPT 的问答交互方式，让门店员工通过自然语言完成日常管理操作，包括核销、预约、收银、数据查询等功能。

## 核心功能

### 🎭 多角色支持

- **前台/收银员**：核销次卡、预约管理、开单收银、办理会员卡、客户查询
- **店长**：经营报表、营业额统计、客户管理、营销活动、库存预警、业绩排名
- **美容师**：排班查询、预约查看、确认到店、客户档案、项目BOM

### 💬 智能对话交互

- 自然语言输入，AI 智能理解意图
- 语音输入支持（长按录音）
- 图片上传识别（名片录入等）
- 结构化卡片展示（确认卡片、数据看板、列表卡片）

### ⚡ 快捷操作

- 每个角色专属的快捷按钮菜单
- 一键发送常用指令
- 自动换行显示，无需横向滚动

### 📊 可视化数据

- 经营数据看板（营业额、客户数、预约完成率）
- 环比趋势分析（上升/下降指示）
- 预约列表、排班表格
- 次卡核销确认流程

## 技术特点

- **前端框架**：React + TypeScript
- **样式方案**：Tailwind CSS v4
- **交互体验**：移动端优先设计（适配 iPhone 14 Pro）
- **响应式布局**：支持手机、平板、桌面端
- **组件化架构**：高度模块化，易于维护扩展

## 设计亮点

### 🎨 视觉风格

- **主色调**：玫瑰金 (#C9956C) + 深紫 (#2D1B69)
- **设计语言**：现代简洁，圆角卡片设计
- **消息气泡**：AI 白色气泡（左对齐）+ 用户玫瑰金渐变气泡（右对齐）
- **卡片样式**：12px 圆角，轻量阴影，浅玫瑰金标题栏

### 📱 交互细节

- 消息自动滚动到底部
- 按钮 hover 状态渐变效果
- 角色切换下拉菜单
- 输入框回车发送
- 快捷按钮点击即发送

## 使用场景

1. **前台收银**：客户到店 → 核销次卡 → 查看预约 → 开单收银
2. **店长管理**：查看今日报表 → 分析营业数据 → 查看业绩排名 → 处理库存预警
3. **美容师服务**：查看今日排班 → 确认客户到店 → 查阅客户档案 → 完成服务项目
