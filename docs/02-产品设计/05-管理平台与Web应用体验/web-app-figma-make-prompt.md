# Figma Make Prompt — 美业智能助手 Web App 原型

## 产品概述

设计一款面向美业门店店长和店员使用的**对话式管理助手 Web App**，采用类 ChatGPT 的问答交互界面，让用户通过自然语言完成门店日常管理操作（查数据、预约、核销、库存等）。

---

## 整体设计风格

- **风格**：现代简洁，偏向移动端 App 风格，参考微信/钉钉的聊天界面
- **主色调**：玫瑰金 `#C9956C` + 深紫 `#2D1B69` + 白色背景
- **字体**：中文优先，正文 14px，标题 16-18px
- **圆角**：卡片 12px，按钮 8px，气泡 16px
- **阴影**：轻量卡片阴影 `0 2px 8px rgba(0,0,0,0.08)`
- **设备**：以 iPhone 14 Pro（390×844px）为主画板，同时提供 iPad 和桌面版

## 组件规范

### 消息气泡
- AI 消息：白色背景，`#E5E7EB` 边框，左上角无圆角（指向左侧）
- 用户消息：玫瑰金渐变（`#C9956C` → `#B8845A`），白色文字，右上角无圆角

### 快捷按钮
- 背景：`#F3F4F6`（浅灰）
- 文字：`#374151`（深灰）
- 边框：`#E5E7EB`
- 激活态：玫瑰金背景 + 白色文字
- 高度：36px，内边距：8px 16px

### 确认卡片
- 背景：白色
- 边框：`#E5E7EB`，1px
- 圆角：12px
- 标题栏：浅玫瑰金背景（`#FDF2EC`）
- 确认按钮：玫瑰金实心（`#C9956C`）
- 取消按钮：灰色描边

### 数据卡片 KPI 格子
- 背景：白色或极浅灰（`#FAFAFA`）
- 数字：24px，粗体，深色（`#111827`）
- 标签：12px，灰色（`#6B7280`）
- 环比上升：绿色（`#10B981`）+ ↑
- 环比下降：红色（`#EF4444`）+ ↓

## 交互标注

在画板上添加以下交互说明标注：

1. 快捷按钮 → 点击后自动填入输入框并发送
2. 确认卡片「确认核销」按钮 → 变为 loading 状态（旋转图标）→ 显示成功 Toast
3. 语音按钮 → 长按进入录音模式（画板 6）
4. 图片按钮 → 弹出选择器（相机/相册/粘贴）
5. 客户列表条目 → 点击展开详情卡片（画板 7）
6. 「查看全部」链接 → 展开完整列表

## 原型流程连接

- 画板 1（登录）→ 点击登录按钮 → 画板 2（前台对话界面）
- 画板 2 → 点击头像 → 角色切换弹窗（可切换到画板 3/4）
- 画板 2 → 点击图片按钮 → 画板 5（图片输入）
- 画板 2 → 长按语音按钮 → 画板 6（录音状态）
- 画板 2 → 点击客户列表条目 → 画板 7（客户详情）
- 画板 8（平板）与画板 2-7 内容相同，仅布局不同
