# 美业管理平台 — 产品说明书

## 1. 产品定位

美业管理平台是一款面向美容院、美容养生会所及美业连锁品牌的一站式数字化经营管理系统。平台覆盖客户管理、智能营销、门店运营、商品管理、订单处理、库存供应链及系统管理等核心业务场景，帮助美业经营者实现精细化运营、智能化决策和高效协同管理。

---

## 2. 目标用户

| 角色 | 使用场景 |
|------|----------|
| 超级管理员 | 全局管控，管理所有门店、用户、角色权限 |
| 门店管理员（店长） | 管理单店日常运营，包括排班、预约、库存、营销 |
| 美容师 | 查看个人排班、预约信息 |
| 收银员 | 处理订单收银、次卡核销 |
| 库存管理员 | 管理产品库存、采购、调拨 |

---

| 模块 | 页面数 | 核心能力 |

## 4. 功能详细说明

### 4.1 仪表盘

数据概览面板，展示核心经营指标和快捷操作入口。

- 核心指标卡片：总客户数、本月新增客户、今日收入、本月总收入，均显示同比变化
- 营销活动概况：进行中活动数、总参与人数、平均转化率，以及近期活动列表
- 订单统计：今日订单数、待处理数、今日预约数
- 热门商品排行：按销量排序的 Top 4 商品
- 快捷操作：一键跳转客户管理、创建营销活动、项目预约、库存管理

### 4.2 客户管理

#### 4.2.1 客户数据
客户信息的集中管理中心。
- 客户列表：支持按姓名、手机号搜索，按会员等级、门店筛选
- 客户信息：姓名、手机号、性别、年龄、会员等级、累计消费、到店次数、标签
- 数据展示：表格视图，支持分页

#### 4.2.2 客户画像
单个客户的 360° 全景视图。
- 基础信息：个人资料、会员等级、消费偏好
- 消费分析：消费趋势、常购项目、消费频次

#### 4.2.3 智能邀约
AI 驱动的客户邀约工具。
- 智能话术生成：根据客户消费记录和偏好，AI 自动生成个性化邀约话术
- 对话式交互：支持多轮对话优化话术内容

### 4.3 智能营销

#### 4.3.1 活动管理
营销活动的全生命周期管理。
- 状态筛选：按「进行中」「即将开始」「已结束」三个标签切换查看
- 活动卡片：展示活动封面图、标题、描述、参与人数、转化率、时间范围、目标客户
- 创建活动弹窗：填写活动名称、目标客户、参与项目/商品、优惠内容、时间范围；支持 AI 文案生成和 AI 海报生成（3 种风格）；支持小程序预览
- 活动详情弹窗：头图展示、核心数据（参与人数/转化率/预估营收/投入成本）、活动信息、近 7 天参与趋势图、最近参与客户列表
- 查看效果：跳转至单个活动的效果分析页

#### 4.3.2 智能推荐
AI 驱动的营销策略推荐引擎。
- 推荐算法说明：综合客户画像、消费习惯、季节特征和行业趋势
- 分类筛选：全部推荐、高转化、客户唤醒、裂变营销、会员关怀
- 推荐卡片：活动图片、标题、标签、匹配度评分、AI 分析原因、目标客户、营销策略、预期效果、活动周期、折扣建议
- 一键创建：点击「立即创建」弹出创建活动弹窗，自动预填推荐数据

#### 4.3.3 策略模板
基于规则的自动化营销策略管理。
- 策略列表：展示策略名称、状态（启用/停用/草稿）、执行方式（自动/手动）、触发规则标签、执行动作标签、覆盖客户数、转化率
- 10 种触发规则：
  - 时间触发：生日关怀、节假日营销（10 个节日可选）、季节性护肤（四季可选）
  - 行为触发：最近消费时间、消费金额、到店频率、产品偏好、沉睡客户唤醒
  - 属性触发：会员等级、新客户引导
- 6 种执行动作：优惠券、折扣、赠品、积分奖励、短信通知、推送通知
- 创建/编辑弹窗：基础信息 + 触发规则配置（可视化参数设置）+ 执行动作配置
- 策略详情弹窗：完整展示规则和效果数据，支持启用/停用切换
- 预置 6 个策略模板：沉睡客户唤醒、生日专属关怀、春季焕肤推荐、高消费客户维护、母亲节感恩活动、新客首次体验

#### 4.3.4 效果分析
营销活动的整体效果数据分析。
- 总览指标：总活动数、进行中活动、总参与人数、平均转化率
- 活动排行：按转化率排序的活动列表，展示参与人数、活动时间、营收、转化率

#### 4.3.5 活动效果（单活动）
单个营销活动的详细效果分析页面。
- 通过活动卡片的「查看效果」按钮进入
- 展示该活动的详细转化漏斗和效果数据

### 4.4 门店管理

#### 4.4.1 项目类型管理
服务项目的分类体系管理。
- 分类列表：支持多级分类展示

#### 4.4.2 项目管理
美容服务项目的 CRUD 管理。
- 项目列表：项目名称、时长、价格、分类
- 添加项目弹窗：填写项目详细信息

#### 4.4.3 美容师管理
美容师团队信息管理。
- 美容师列表：姓名、手机号、等级、专长、状态、所属门店
- 支持新增、编辑美容师信息

#### 4.4.4 美容师等级设置
美容师等级体系配置。
- 等级列表：等级名称、分数区间、提成比例、描述
- 支持等级的增删改

#### 4.4.5 排班管理
美容师工作排班日历。
- 周视图：按周展示 7 天排班，分上午/下午时段
- 美容师切换：Tab 切换不同美容师的排班
- 时段状态：可预约（绿色）/ 不可预约（灰色）
- 支持按天/按周视图切换

#### 4.4.6 项目预约
客户服务预约管理。
- 预约列表：客户、项目、美容师、门店、日期时间、状态
- 预约状态：待确认、已确认、进行中、已完成、已取消

### 4.5 商品管理

#### 4.5.1 商品类型
商品分类的树形结构管理。
- 分类树：一级分类（护肤品、美发产品、美甲产品、仪器耗材、日用消耗品）+ 二级子分类
- 详情面板：选中分类后显示描述、层级、关联商品数、子分类数
- 支持新增一级分类、添加子分类、编辑、删除（仅无关联商品时可删）

#### 4.5.2 商品管理
商品信息的完整管理。
- 左侧分类树 + 右侧商品表格联动
- 商品信息：缩略图、名称、SKU（自动生成，格式 SK-LO-000001）、品牌、规格、单位、成本价、零售价、保质期、供应商、状态
- 操作：编辑、详情、停售
- 添加商品弹窗：完整字段录入，SKU 自动生成

#### 4.5.3 次卡管理
次卡套餐的管理。
- 次卡列表：名称、类型、总次数、价格、有效期、关联项目、状态
- 支持次卡的创建和编辑

### 4.6 订单管理

#### 4.6.1 商品订单管理
商品销售订单的管理。
- 统计卡片：总订单数、已完成、待处理、总金额
- 筛选：按订单号/客户搜索，按状态（待付款/已付款/已完成/已取消/已退款）和门店筛选
- 订单表格：订单编号、客户（姓名+电话）、门店、商品数、总金额、支付方式、状态、下单时间
- 订单详情弹窗：订单信息 + 商品明细表 + 合计金额

#### 4.6.2 次卡开卡管理
次卡购买订单管理。
- 开卡记录列表：订单编号、客户、卡名、总次数、剩余次数、金额、状态、购买日期、到期日期

#### 4.6.3 次卡核销管理
次卡使用核销记录。
- 核销记录列表：核销详情、关联次卡、使用项目

### 4.7 库存管理

#### 4.7.1 产品管理
库存产品目录管理（与商品管理共用页面，从库存视角访问）。

#### 4.7.2 库存管理
库存总览与批次管理。
- 筛选：门店、分类、状态（正常/低库存/积压/缺货）、搜索
- 库存表格：产品名称、SKU、当前库存、已预留、可用库存、安全库存、最大库存、状态标签、最近入库日期
- 批次详情面板（右侧滑出）：批次号、入库量、可用量、生产日期、过期日期、状态、入库日期，按 FIFO 排序
- 入库弹窗：产品选择、入库数量、批次号（自动生成）、生产日期、过期日期、供应商、关联采购订单

#### 4.7.3 采购管理
AI 补货建议与采购订单全流程。
- 补货建议 Tab：
  - AI 预测提示（基于近 90 天数据，置信度 85%）
  - 建议表格：产品、SKU、当前库存、预测需求(7天)、安全库存、在途数量、建议补货量（可编辑）、供应商、预估金额
  - 批量勾选 + 生成采购订单（按供应商分组）
- 采购订单 Tab：
  - 订单表格：编号、供应商、门店、产品数、总金额、状态、创建日期、预计交货日期
  - 订单状态流转：草稿 → 待审核 → 已审核 → 已下单 → 已收货 / 已取消
  - 订单详情弹窗：订单信息 + 产品明细 + 合计金额 + 状态操作按钮

#### 4.7.4 过期管理
临期预警与损耗分析。
- 统计卡片：临期预警数（<60天）、紧急处理数（<30天）、本月损耗金额（含环比）
- 临期产品表格：紧急程度标签（临期/紧急/已过期）、产品信息、批次号、剩余天数、库存量、成本金额、门店、建议处置（促销/调拨/报废）
- 损耗分析：近 6 月损耗金额柱状图 + 按品类损耗占比图 + 损耗率超标类别（连续 3 月 >5% 标红）

#### 4.7.5 门店库存与调拨
多门店库存对比与智能调拨。
- 门店卡片（横向滚动）：门店名、地址、SKU 数、库存总值、健康度评分、采购模式
- 库存对比 Tab：矩阵表格，行为产品，列为各门店库存数量，色块标状态（绿正常/橙偏低/红缺货）
- 调拨管理 Tab：
  - AI 调拨建议：调出店、调入店、产品、建议量、原因
  - 调拨单列表：编号、调出店、调入店、产品数、状态（待确认/运输中/已完成/已取消）
  - 发起调拨弹窗：选择调出/调入门店、添加产品明细、备注

#### 4.7.6 服务消耗与 BOM
服务项目物料清单与消耗追踪。
- BOM 管理 Tab：服务项目可展开列表，展示 BOM 明细（产品、SKU、标准用量、单位），支持编辑 BOM
- 消耗记录 Tab：日期、服务项目、客户、美容师、门店、产品、标准用量、实际用量、偏差%（>20% 红色高亮异常标记），支持按日期/门店/美容师/异常筛选
- 库存预估 Tab：未来 7 天预约消耗汇总（产品、预计消耗、当前库存、缺口量红色标注），缺口产品关联补货建议

### 4.8 系统设置

#### 4.8.1 用户管理
系统用户的集中管理。
- 统计卡片：总用户数、已启用、已禁用、角色数
- 筛选：按用户名/姓名/手机号搜索，按角色和状态筛选
- 用户表格：用户名、姓名、手机号、角色（带颜色标签）、所属门店、状态、最后登录时间
- 操作：编辑、启用/禁用、删除（超级管理员不可删）
- 新增/编辑弹窗：用户名、姓名、手机号、邮箱、初始密码（仅新增）、角色选择、门店分配

#### 4.8.2 角色管理
RBAC 角色与权限配置。
- 左侧角色卡片列表：角色名、编码、描述、用户数、系统角色标识
- 右侧权限详情面板：按模块分组展示权限配置（全部权限/部分权限/无权限）
- 预置 5 个角色：超级管理员、门店管理员、收银员、美容师、库存管理员
- 新增/编辑弹窗：角色名称、编码、描述 + 按模块勾选权限（支持全选/单选）

#### 4.8.3 权限管理
系统权限点的结构化展示。
- 统计：菜单权限数、操作权限数、权限总数
- 权限树：8 大模块（仪表盘、客户管理、智能营销、门店管理、商品管理、订单管理、库存管理、系统设置），每个模块下包含细粒度操作权限
- 权限类型：菜单（蓝色）、操作（绿色）
- 详情面板：权限编码、类型、描述、子权限数

#### 4.8.4 门店管理
门店信息的全局管理。
- 统计卡片：门店总数、营业中、总员工数、月总营收
- 门店卡片：门店名称、编码、地址、电话、营业时间、店长、采购模式、员工数/SKU数/月营收
- 门店状态：营业中、装修中、已关闭
- 新增/编辑弹窗：门店名称、编码、地址、电话、店长、营业时间、采购模式（集中/独立）、状态
- 详情弹窗：完整门店信息 + 员工数/SKU数/月营收数据卡片

## 7. 数据架构

### 7.1 API 层
已建立统一的 API 抽象层（`src/api/`），按模块封装：
- `client.ts`：Axios 实例，含 JWT 认证拦截器和统一错误处理
- `product.ts`：产品和分类 CRUD
- `inventory.ts`：库存、批次、临期、补货建议、采购订单
- `customer.ts`：客户查询和筛选
- `order.ts`：商品订单查询

### 7.2 类型定义
统一类型层（`src/types/`），覆盖 8 个业务域：
- 客户（Customer、CustomerTag）
- 产品（Product、Category）
- 库存（StockItem、Batch、ExpiringProduct、ReplenishmentSuggestion、PurchaseOrder）
- 门店（Store、Beautician、BeauticianLevel、Schedule）
- 订单（ProductOrder、Reservation、Card、CardOrder）
- 营销（MarketingActivity、MarketingTemplate、MarketingRecommendation）
- 调拨（StockComparison、TransferSuggestion、TransferOrder）
- BOM（Service、BOMItem、ConsumptionRecord、ForecastItem）
