# 美丽人生 - 美容院预约小程序

## 项目简介

这是一个基于HTML5的美容院线上预约小程序，提供完整的预约服务、AI美容助手、美容课堂和用户管理功能。

## 功能特性

### 🏠 首页
- 品牌展示和介绍
- 服务项目展示（面部护理、身体按摩、美体塑形、美甲服务）
- AI美容助手入口
- 美容课堂入口

### 📅 预约服务
- 服务项目选择
- 日期和时间选择
- 美容师选择
- 预约信息确认
- 实时价格计算

### 🤖 AI美容助手
- 智能美容咨询
- 常见问题快速回答
- 个性化美容建议
- 实时对话交互

### 🎓 美容课堂
- 护肤知识课程
- 化妆技巧教程
- 身体护理指南
- 美容趋势分享

### 📋 订单管理
- 订单状态跟踪
- 预约历史查看
- 订单状态筛选
- 订单操作管理

### 👤 个人资料
- 用户信息管理
- 皮肤类型设置
- 美容关注点配置
- 头像上传功能

## 技术架构

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **UI框架**: 原生CSS，响应式设计
- **图标库**: Font Awesome 6.0
- **存储**: LocalStorage
- **兼容性**: 支持现代浏览器和移动端

## 开发说明

### 添加新服务
在 `script.js` 中的 `servicePrices` 对象添加新服务：

```javascript
const servicePrices = {
    'new-service': 150,
    // ... 其他服务
};
```

### 添加新美容师
在 `script.js` 中的 `beauticians` 对象添加新美容师：

```javascript
const beauticians = {
    'new-beautician': {
        name: '新美容师',
        title: '职位',
        experience: '经验',
        rating: 5.0,
        avatar: '头像URL'
    }
};
```

### 自定义样式
修改 `styles.css` 文件中的CSS变量来调整主题色彩：

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #ff6b6b;
}
```
