# 数据字典

更新时间：2026-06-01

## 范围

本文记录本次数据治理与经营闭环开发新增和关键串联的数据表，重点覆盖库存、订单、营销归因、终端推荐、促销和打印任务。

## 新增核心表

### StockMovement 库存流水

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| id | 主键 | 自增 |
| storeId | 门店 ID | 对应 `Store.id` |
| productId | 商品 ID | 对应 `Product.id` |
| batchId | 批次 ID | 可为空，对应 `StockBatch.id` |
| movementNo | 流水号 | 唯一业务编号 |
| movementType | 流水类型 | `inbound`、`sale_out`、`service_consume`、`transfer_out`、`transfer_in` |
| quantity | 变动数量 | 入库为正，出库/消耗为负 |
| beforeStock | 变动前库存 | 冗余快照，便于审计 |
| afterStock | 变动后库存 | 冗余快照，便于审计 |
| sourceType | 来源类型 | `stock_batch`、`product_order`、`service_task`、`consumption_record`、`transfer_order` |
| sourceId | 来源 ID | 对应来源业务表主键 |
| sourceNo | 来源编号 | 订单号、任务号、批次号等 |
| occurredAt | 发生时间 | 默认当前时间 |

### OrderItem 订单明细

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| orderId | 订单 ID | 对应 `ProductOrder.id` |
| itemType | 明细类型 | `product`、`project`、`card`、`recharge` |
| itemId | 业务对象 ID | 商品、项目、卡项等 |
| name | 明细名称 | 展示与审计使用 |
| quantity | 数量 | Decimal |
| unitPrice | 单价 | Decimal |
| subtotal | 小计 | Decimal |
| discount | 优惠 | 默认 0 |
| payload | 原始载荷 | 保留终端/管理端提交快照 |

### PaymentRecord 支付记录

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| orderId | 订单 ID | 对应 `ProductOrder.id` |
| paymentNo | 支付流水号 | 唯一业务编号 |
| method | 支付方式 | `wechat`、`alipay`、`cash`、`card`、`customer_card` 等 |
| amount | 支付金额 | Decimal |
| status | 支付状态 | 当前写入 `success` |
| transactionNo | 三方交易号 | 可为空 |
| paidAt | 支付时间 | 可为空 |

### RefundRecord 退款记录

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| orderId | 订单 ID | 对应 `ProductOrder.id` |
| refundNo | 退款流水号 | 唯一业务编号 |
| amount | 退款金额 | Decimal |
| reason | 退款原因 | 可为空 |
| status | 退款状态 | 当前写入 `success` |
| refundedAt | 退款时间 | 可为空 |

### MarketingAttribution 营销归因

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| touchId | 触达 ID | 对应 `MarketingAutomationTouch.id` |
| strategyId | 策略 ID | 对应 `MarketingAutomationStrategy.id` |
| executionId | 执行 ID | 对应 `MarketingAutomationExecution.id` |
| customerId | 客户 ID | 对应 `Customer.id` |
| orderId | 订单 ID | 对应 `ProductOrder.id` |
| attributionType | 归因类型 | 当前使用 `last_touch` |
| attributedRevenue | 归因收入 | 订单完成写入，退款时冲减 |
| attributionWindowDays | 归因窗口 | 默认 30 天 |
| occurredAt | 归因发生时间 | 订单支付/完成时间 |

### RecommendationEvent 推荐反馈

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| storeId | 门店 ID | 对应 `Store.id` |
| customerId | 客户 ID | 对应 `Customer.id` |
| deviceId | 终端设备 ID | 对应 `TerminalDevice.id` |
| recommendationId | 推荐 ID | 可为空 |
| eventType | 事件类型 | `view`、`accept`、`reject`、`feedback` 等 |
| taskId | 服务任务 ID | 可为空 |
| orderId | 订单 ID | 可为空 |
| note | 备注 | 可为空 |
| payload | 原始载荷 | 保留端侧上下文 |

### Promotion 活动权益

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| storeId | 门店 ID | 可为空，为空表示全门店 |
| name | 活动名称 | 必填 |
| description | 活动说明 | 可为空 |
| discountText | 优惠文案 | 必填 |
| applicableProjectIds | 适用项目 | PostgreSQL Int 数组 |
| startAt | 开始时间 | 可为空 |
| endAt | 结束时间 | 可为空 |
| status | 状态 | `active` 生效 |

### PrintJob 打印任务

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| storeId | 门店 ID | 对应 `Store.id` |
| jobNo | 打印任务号 | 唯一业务编号 |
| sourceType | 来源类型 | `order`、`card`、`recharge`、`custom` 等 |
| sourceId | 来源 ID | 可为空 |
| title | 打印标题 | 必填 |
| content | 打印内容 | 必填 |
| copies | 打印份数 | 默认 1 |
| status | 状态 | `pending`、`completed`、`failed` |
| errorMessage | 错误信息 | 可为空 |
| completedAt | 完成时间 | 可为空 |

## 主链路写入规则

| 业务动作 | 写入表 | 说明 |
| --- | --- | --- |
| 管理端入库 | `StockBatch`、`StockMovement`、`Product.currentStock` | 批次和库存总量同步增加 |
| 终端完成服务 | `ServiceTask`、`StockMovement`、`Product.currentStock` | `consumptionItems` 生成服务消耗流水 |
| 终端收银 | `ProductOrder`、`OrderItem`、`PaymentRecord`、`ConsumptionRecord`、`MarketingAttribution` | 商品明细会额外生成 `sale_out` 库存流水 |
| 终端办卡 | `CustomerCard`、`ProductOrder`、`OrderItem`、`PaymentRecord`、`MarketingAttribution` | 卡项成为客户资产 |
| 终端充值 | `ProductOrder`、`OrderItem`、`PaymentRecord`、`ConsumptionRecord`、`MarketingAttribution` | 保留充值与赠送金额 |
| 订单退款 | `RefundRecord`、`ProductOrder.status`、`MarketingAttribution` | 归因收入和客户累计消费同步冲减 |
| 推荐反馈 | `RecommendationEvent` | 绑定门店、客户、设备、任务或订单 |
| 打印小票 | `PrintJob` | 可按来源追踪打印状态 |
