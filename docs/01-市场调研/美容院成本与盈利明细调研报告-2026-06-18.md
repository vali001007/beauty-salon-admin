# 美容院成本与盈利明细调研报告

调研日期：2026-06-18  
调研对象：中国大陆生活美容/皮肤管理/身体护理门店，兼顾头部连锁公开财报与中小门店经营模型。  
结论口径：以下测算以人民币计，默认讨论“单店经营收入”而非预收卡现金流；预收卡能改善现金流，但不能直接等同利润。

## 1. 核心结论

1. 美容院不是简单的“高毛利生意”，真实净利取决于房租、人效、复购和项目结构。头部连锁 2025 年上半年综合毛利率约 49.3%，经调整净利率约 13.1%；其中“美容与保健服务”直营店毛利率约 40.7%。这说明即使是强管理连锁，扣除人工、租金折旧、营销和总部费用后，净利也不是传统宣传里的 40%-50%。
2. 中小美容院的合理净利区间通常应按 5%-20% 评估：新店爬坡期常见亏损或盈亏平衡；稳定社区店做到 8%-15% 已经有经营质量；项目结构好、复购强、房租低、人效高的门店可到 15%-25%。超过 30% 的持续净利需要谨慎核验，通常依赖老板亲自服务、不计老板工资、强预收、低租金或高客单高转化。
3. 单店盈利关键不是“卖卡金额”，而是四个指标：有效到店客流、客单价、复购频次、项目毛利。卖卡只解决现金流，不解决履约成本；如果项目设计不清、库存耗材不准、服务排班低效，预收越多，后期履约压力越大。
4. 成本大头一般是人工、房租/物业、耗材/产品、营销获客、装修设备折旧。对中小店来说，最需要数字化管理的是项目毛利、耗材扣减、员工提成、会员卡剩余权益、复购提醒和客户生命周期价值。
5. 生活美容与医美要分开看：生活美容依赖手工服务和复购，毛利受人工强约束；医美非手术/轻医美客单更高，但合规、医生/护士、器械药品、获客成本和风险准备金也明显更高。

## 2. 公开数据基准

### 2.1 头部连锁样本：美丽田园

美丽田园医疗健康产业有限公司是港股上市公司，公开资料可作为高端连锁美业的真实财务锚点。其 2025 年上半年业绩材料披露：

| 指标 | 2025H1 数值 | 对美容院经营的含义 |
| --- | ---: | --- |
| 总收入 | 14.59 亿元 | 头部连锁规模化后仍依赖多业务线组合 |
| 综合毛利 | 7.20 亿元 | 综合毛利率约 49.3% |
| 经调整净利润 | 1.91 亿元 | 经调整净利率约 13.1% |
| 美容与保健服务收入占比 | 55% | 生活美容/保健是基础收入盘 |
| 医疗美容服务收入占比 | 34% | 高客单业务拉高综合毛利 |
| 亚健康医疗服务收入占比 | 11% | 健康管理与调理类增长业务 |
| 美容与保健服务直营店毛利率 | 40.7% | 更接近生活美容直营门店履约口径 |
| 美容与保健服务加盟店毛利率 | 56.4% | 加盟口径与直营单店经营成本不同，不宜直接套用 |
| 医疗美容服务毛利率 | 56.9% | 医美毛利更高，但合规、人力、药械和获客风险也更高 |
| 亚健康医疗服务毛利率 | 63.1% | 样本为头部连锁，不代表普通中小美容院 |

美丽田园 2025H1 的营业成本约 7.39 亿元，成本内部结构中，员工成本、折旧摊销、已用产品及耗用品占主要部分。按业绩 PPT 披露的营业成本构成拆解，约可理解为：

| 营业成本项目 | 在营业成本中占比 | 折算占收入比例（按总收入 14.59 亿元估算） | 经营解释 |
| --- | ---: | ---: | --- |
| 折旧摊销 | 约 35% | 约 17.7% | 门店装修、仪器设备、使用权资产等长期投入 |
| 员工成本 | 约 30% | 约 15.2% | 美容师、店务、服务人员等直接履约成本 |
| 已用产品及耗用品 | 约 24% | 约 12.2% | 项目耗材、院线产品、一次性用品等 |
| 运营相关成本 | 约 10% | 约 5.1% | 门店运营、维修、能源等 |
| 其他 | 约 1% | 约 0.5% | 零散履约成本 |

注意：上市连锁的折旧摊销和总部费用口径与街边单店不同，但它的价值在于提示：美容院的真实利润会被履约人工、门店固定资产和耗材持续吞噬。

### 2.2 消费环境与渠道趋势

国家统计局 2025 年数据表明，全国居民人均消费支出 29,476 元，同比增长 4.4%；其中人均服务性消费支出 13,602 元，同比增长 4.5%，占居民人均消费支出的 46.1%。这对美容院是利好：生活服务消费仍在增长，但增长不意味着单店自然盈利，门店仍要争夺本地服务消费预算。

美团 2025 年生活服务趋势数据显示，线下体验消费继续回流，“疗愈悦己”类关键词搜索和相关商户供给均有明显增长，精油 SPA 线上交易同比增长超过 74%。这说明美容院可以承接“情绪疗愈、压力释放、轻护理”的需求，但消费者也更关注性价比和平替，单纯高价办卡的转化难度上升。

毕马威 2025 中国美妆市场报告显示，美妆消费的线上渠道继续强势，2025 年第一季度主要线上平台化妆品零售额同比增长，美容护肤品类占比较高。对美容院的影响是：零售产品利润空间被电商透明化压缩，门店产品销售不能只靠信息差，需要绑定专业检测、护理方案和复购服务。

## 3. 美容院收入结构

美容院的收入一般分为六类。做经营分析时，应把“现金收入”和“履约收入”分开记录。

| 收入类型 | 常见项目 | 毛利特点 | 风险点 |
| --- | --- | --- | --- |
| 基础面部护理 | 清洁、补水、修护、舒敏、痘肌护理 | 客单中低、复购强、耗材可控 | 同质化强，容易陷入低价团购 |
| 身体护理/SPA | 精油 SPA、肩颈、经络、塑形、排湿 | 依赖手工服务，人力占比高 | 技师排班和服务时长决定人效 |
| 仪器类项目 | 小气泡、光电类、抗衰仪器、皮肤检测后护理 | 客单较高，折旧和合规要求更高 | 仪器闲置会拖累利润 |
| 套卡/会员卡 | 次卡、疗程卡、储值卡、年卡 | 改善现金流，锁定复购 | 如果权益设计不清，后期履约亏损 |
| 产品零售 | 院线护肤、修护套盒、居家护理 | 理论毛利高，但受电商比价压制 | 库存积压、临期、员工强推伤复购 |
| 增值服务 | 皮肤检测、咨询、私域陪伴、节日活动 | 能提升转化和客单 | 需要标准化话术和客户画像 |

经营上建议至少拆出以下指标：

- 到店人数：新客、老客、沉睡唤醒客分开看。
- 客单价：按服务收入、产品收入、办卡现金流、真实消课收入分开看。
- 消课率：预收卡是否正在被健康履约。
- 项目毛利：每个项目的耗材、服务时长、技师提成、仪器折旧。
- 复购周期：面护 14-30 天，身体护理 7-21 天，抗衰/仪器类按疗程节奏管理。
- 技师人效：每人月服务收入、有效工时、到店转化率、续卡率。

## 4. 一次性开店成本

以下是基于公开开店资料、门店经营访谈口径和美业常见投入项整理的区间，适合用于商业计划初算。实际成本与城市、商圈、面积、装修标准、加盟/自营、仪器配置差异很大。

| 成本项 | 小型工作室/社区轻护理（30-60㎡） | 标准社区美容院（80-150㎡） | 中高端会所/皮肤管理中心（150-300㎡） |
| --- | ---: | ---: | ---: |
| 房租押金/预付 | 2-8 万 | 6-20 万 | 20-80 万 |
| 装修设计 | 3-10 万 | 10-35 万 | 40-150 万 |
| 美容床/基础设备 | 1-5 万 | 5-15 万 | 15-40 万 |
| 仪器设备 | 1-10 万 | 10-50 万 | 50-200 万以上 |
| 首批产品/耗材 | 1-5 万 | 5-20 万 | 20-80 万 |
| 收银/系统/监控 | 0.5-3 万 | 2-8 万 | 5-20 万 |
| 开业营销 | 0.5-5 万 | 3-15 万 | 10-50 万 |
| 人员招聘培训 | 0.5-3 万 | 2-8 万 | 5-20 万 |
| 流动资金 | 3-10 万 | 10-30 万 | 30-100 万 |
| 合计参考 | 12-59 万 | 53-201 万 | 195-740 万以上 |

判断建议：

- 小店不要把预算过度砸在装修上，优先保证服务项目、耗材品质、私域转化和复购机制。
- 标准店要预留 3-6 个月流动资金，因为新店从开业到稳定复购通常有爬坡期。
- 高端店若依赖仪器和会所环境，需要重点测算设备利用率和单房间产出，否则折旧压力会很大。

## 5. 月度运营成本明细

按标准社区店（80-150㎡、3-6 名服务人员）估算，月度成本结构一般如下：

| 成本项 | 常见占收入比例 | 月收入 15 万示例 | 管理重点 |
| --- | ---: | ---: | --- |
| 产品/耗材/一次性用品 | 10%-22% | 1.5-3.3 万 | 项目 BOM、自动扣耗材、临期管理 |
| 美容师工资与提成 | 22%-35% | 3.3-5.25 万 | 技师人效、排班、提成规则 |
| 店长/前台/运营人员 | 5%-12% | 0.75-1.8 万 | 店长续卡和复购责任要量化 |
| 房租/物业 | 8%-20% | 1.2-3 万 | 超过 18% 后盈利压力明显上升 |
| 水电/洗涤/维修 | 2%-5% | 0.3-0.75 万 | 仪器、空调、热水、布草管理 |
| 获客营销 | 5%-15% | 0.75-2.25 万 | 团购、达人、老带新、私域活动 ROI |
| 软件/支付/办公 | 1%-3% | 0.15-0.45 万 | 会员、库存、收银、短信等 |
| 装修/设备折旧 | 4%-12% | 0.6-1.8 万 | 仪器利用率和项目定价 |
| 税费/手续费/杂费 | 2%-6% | 0.3-0.9 万 | 发票、支付费率、财务合规 |
| 老板工资/管理预留 | 5%-10% | 0.75-1.5 万 | 不计老板工资会高估净利 |

经验判断：

- 人工 + 房租合计最好控制在收入的 45% 以内；超过 55% 时，除非客单价高或产品毛利强，否则盈利很难稳定。
- 产品耗材不是越低越好，过低可能牺牲体验和效果；关键是每个项目都能算清“耗材成本 + 服务时长 + 提成 + 折旧”。
- 营销费要按渠道 ROI 管理：团购低价引流、达人种草、老客裂变、会员唤醒的回本周期不同，不能混在一个“推广费”里。

## 6. 单店盈利测算模型

### 6.1 标准社区美容院：基准模型

假设：120㎡，4 名美容师、1 名前台/店助、1 名店长；月收入 15 万元，收入以真实消课和服务收入计，不把未履约预收卡一次性计为利润。

| 项目 | 金额/月 | 占收入比例 |
| --- | ---: | ---: |
| 服务收入 | 12.0 万 | 80% |
| 产品零售收入 | 2.0 万 | 13% |
| 其他增值收入 | 1.0 万 | 7% |
| 总收入 | 15.0 万 | 100% |
| 产品/耗材/进货成本 | -2.5 万 | -16.7% |
| 美容师工资与提成 | -3.9 万 | -26.0% |
| 店长/前台/社保等 | -1.2 万 | -8.0% |
| 房租/物业 | -1.8 万 | -12.0% |
| 水电维修/布草/办公 | -0.7 万 | -4.7% |
| 营销获客 | -1.2 万 | -8.0% |
| 系统/支付/短信/工具 | -0.25 万 | -1.7% |
| 折旧摊销 | -0.8 万 | -5.3% |
| 税费/杂费/坏账预留 | -0.55 万 | -3.7% |
| 老板工资或管理预留 | -0.8 万 | -5.3% |
| 经营净利润 | 1.3 万 | 8.7% |

这个模型说明：月入 15 万的社区美容院，如果规范计入老板工资、折旧、营销和耗材，净利可能只有 8%-12%。如果老板亲自服务、不计工资，账面净利会提高，但不代表商业模型更强。

### 6.2 盈亏平衡模型

简化公式：

```text
盈亏平衡月收入 = 固定成本 / (1 - 变动成本率)
```

示例：

| 参数 | 保守店 | 基准店 | 优秀店 |
| --- | ---: | ---: | ---: |
| 月固定成本 | 6.5 万 | 7.5 万 | 9.5 万 |
| 变动成本率 | 38% | 34% | 30% |
| 盈亏平衡月收入 | 约 10.5 万 | 约 11.4 万 | 约 13.6 万 |
| 稳定盈利所需收入 | 12-14 万 | 15-18 万 | 20 万以上 |

固定成本包括房租、基础工资、前台/店长、系统、折旧、基础水电、老板工资预留等。变动成本包括项目耗材、产品进货、技师提成、部分营销费和支付手续费。

### 6.3 不同经营水平的利润表现

| 类型 | 月收入 | 净利率 | 月净利润 | 状态判断 |
| --- | ---: | ---: | ---: | --- |
| 新店爬坡/低复购 | 5-8 万 | -20%-5% | 亏损至微利 | 客流不足，固定成本压垮利润 |
| 普通社区店 | 10-15 万 | 3%-10% | 0.3-1.5 万 | 靠老板盯店，有利润但不轻松 |
| 稳定成熟店 | 15-25 万 | 8%-18% | 1.2-4.5 万 | 复购稳定，项目和排班较健康 |
| 优秀单店 | 25-40 万 | 15%-25% | 3.75-10 万 | 客单、复购、人效和毛利同时优秀 |
| 高端/医美化门店 | 40 万以上 | 波动大 | 波动大 | 高客单伴随高获客、合规和人才成本 |

## 7. 影响利润的关键变量

### 7.1 客单价

美容院常见客单分层：

- 低价引流：39-199 元，适合新客体验，不宜长期作为主收入。
- 基础护理：199-499 元，承担复购和会员转化。
- 疗程项目：1,000-5,000 元/疗程，考验效果承诺和履约能力。
- 仪器/抗衰：单次或套餐价格更高，但折旧、合规和客户预期更强。
- 产品套盒：300-2,000 元不等，受品牌、成分、渠道价格透明度影响。

客单价提高不一定提升利润。如果高客单来自高折扣套餐或高提成强销售，毛利可能被折价、赠送项目和员工提成抵消。

### 7.2 复购与消课

美容院盈利的本质是复购生意。相比单次新客，老客复购的获客成本低、信任更高、项目设计更连续。要重点看：

- 首次体验后 7 日内二次预约率。
- 30/60/90 天复购率。
- 会员卡消课率和剩余权益。
- 沉睡客户唤醒率。
- 复购项目是否从低价护理升级到疗程或居家产品。

### 7.3 人效与房间效率

美容院人工强、空间强，不能只看总收入。建议至少看：

- 美容师月服务收入。
- 美容师有效服务小时数。
- 单房间日均服务小时数。
- 每个项目占用床位时长和毛利。
- 高峰期排班利用率。

一个常见问题是：门店看起来客流不少，但大量低价长时项目占用床位和技师，导致高毛利项目排不上，最终利润偏低。

### 7.4 项目毛利

建议每个项目记录以下字段：

```text
项目售价
- 耗材成本
- 产品成本
- 技师提成
- 仪器折旧
- 房间/床位占用成本
= 项目贡献毛利
```

项目贡献毛利低于 40% 时，要么作为引流项目，要么需要优化耗材、时长、定价或套餐结构。不能让低贡献项目成为主收入。

## 8. 常见利润陷阱

1. 把预收卡当利润：现金到账后还有未来服务、耗材、提成和退款风险。
2. 不计老板工资：老板全天盯店、亲自服务，账面利润高，但复制性低。
3. 不计折旧：装修和仪器一次性投入如果不按月摊销，会高估利润。
4. 赠送项目过多：办卡赠送、节日赠送、老客补偿都会形成隐性履约成本。
5. 库存不清：院线产品、精油、面膜、一次性耗材积压或流失，会直接吞利润。
6. 低价团购依赖：新客多但复购低，营销费和服务成本消耗后难盈利。
7. 提成规则失控：只按销售额提成，容易强推卡项；只按服务提成，可能不愿转化。
8. 项目不分层：所有客户都推同一套高价项目，会损害信任和复购。
9. 店长只管人不管数：没有日清、周复盘、月毛利表，问题发现太晚。

## 9. 对 Ami 产品的启发

围绕美容院成本与盈利，Ami 管理系统最有价值的经营模块应优先服务“利润看得清、复购抓得住、履约不亏钱”。

### 9.1 必须做清的经营看板

- 月收入：现金收入、消课收入、产品收入分开。
- 毛利：按项目、员工、客户、门店维度拆。
- 成本：耗材、人工、房租、营销、折旧、杂费。
- 预收负债：会员卡余额、剩余次数、赠送权益。
- 复购漏斗：新客体验、二次到店、办卡、消课、续卡、沉睡。
- 技师人效：服务收入、续卡贡献、好评、消课完成率。

### 9.2 最能降本的功能

- 项目 BOM：每个服务项目绑定标准耗材和用量。
- 服务完成自动扣库存：减少人工录入和耗材黑洞。
- 智能排班：按预约、技师能力、房间资源和项目时长排班。
- 会员卡履约预警：余额过高、消课过慢、即将到期、异常赠送。
- 营销 ROI：每次活动追踪到店、办卡、复购和毛利，而不只看报名数。

### 9.3 最能增收的功能

- 客户画像：肤质、项目偏好、消费力、复购周期、敏感问题。
- 智能推荐：下一次护理、居家产品、疗程续购、沉睡唤醒。
- 私域自动化：护理后回访、复购提醒、生日/节日权益、疗程进度提醒。
- 美容师终端：服务前查看客户档案，服务后记录效果和推荐下一步。

## 11. 后续建议

如果要把这份调研转成产品功能，建议下一步输出《美容院经营利润看板需求文档》，核心包括：

1. 单店利润表：收入、成本、毛利、净利。
2. 项目毛利表：项目售价、耗材、时长、提成、折旧、贡献毛利。
3. 会员卡履约表：预收、消课、剩余权益、到期风险。
4. 员工人效表：服务收入、续卡率、复购率、客户满意度。
5. 库存耗材表：服务自动扣耗、低库存提醒、异常消耗预警。
