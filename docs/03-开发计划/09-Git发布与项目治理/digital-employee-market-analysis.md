## 一、国内美业 SaaS 竞品定价与 AI 功能现状

### 1.1 主要竞品概览

| 竞品 | 定位 | 核心定价模式 | AI 能力 |
|------|------|-------------|---------|
| **博卡** | 美业门店管理系统（头部） | 年费 3000-20000/店/年，按版本分层 | 会员分析、智能排班、营销建议（规则驱动为主） |
| **美团美业管家** | 美团生态内门店管理 | 佣金抽成（团购订单 8-15%）+ 增值服务费 | 客流预测、智能推荐（依托美团大数据） |
| **有赞美业** | 私域运营 + 门店管理 | SaaS 年费 6800-58000 + 交易抽成 0.6% | 智能标签、自动化营销、AI 文案生成 |
| **美盈易** | 中小美容院管理 | 年费 2000-8000/店/年 | 基础会员分析、预约提醒 |
| **美务通** | 美发美容连锁 | 年费 3000-15000/店/年 | 员工绩效分析、客户流失预警 |
| **悦客** | 美业 + 健康行业 | 年费 + 增值模块购买 | AI 客服、智能推荐 |
| **艾优你** | 皮肤管理 SaaS | 年费 5000-20000 | 皮肤检测分析、项目推荐 |

### 1.2 定价模式分析

**主流模式仍为"固定年费/月费"**：

- **按版本分层**：基础版（客户+预约+收银）→ 专业版（+ 营销 + 报表）→ 旗舰版（+ 连锁 + API）
- **按门店数量**：连锁品牌按门店数 × 单店费用
- **交易抽成补充**：有赞、美团会在订单中叠加 0.6-15% 的交易手续费/佣金

**尚无竞品采用"数字员工/工时+提成"模式**——这是一个空白定位机会。

### 1.3 AI 功能现状

当前美业 SaaS 的 AI 能力普遍处于**规则+基础模型**阶段：

| 能力层级 | 代表功能 | 谁在做 |
|----------|----------|--------|
| L1 规则自动化 | 生日提醒、到期通知、自动排班 | 几乎所有竞品 |
| L2 数据分析 | RFM 分群、流失预警、消费趋势 | 博卡、有赞、美团 |
| L3 AI 生成 | 营销文案生成、邀约话术 | 有赞（接入通义千问）、少数新兴玩家 |
| L4 意图驱动交互 | 自然语言操作、智能终端 | **目前无竞品达到此水平** |

**结论**：Ami_Aura 的意图驱动终端 + AI 推荐闭环在美业 SaaS 领域没有直接对标产品。

## 二、"AI 数字员工"先行者的收费模式

### 2.1 数字人/虚拟员工赛道

| 厂商 | 产品形态 | 收费模式 | 参考价格 |
|------|----------|----------|----------|
| **百度曦灵** | 数字人直播/客服 | 按分钟计费 + 定制费 | 数字人直播 0.5-3元/分钟，形象定制 5-50 万 |
| **硅基智能** | 数字人直播带货 | 基础费 + GMV 提成（1-5%） | 月费 3000-10000 + 带货佣金 |
| **腾讯云智能客服** | AI 客服机器人 | 按会话/按并发 | 0.01-0.05元/轮对话，或 500-5000/坐席/月 |
| **小冰/小Ice** | 数字员工（多场景） | 订阅 + 按任务计费 | 月费 + 超量按次 |
| **Salesforce Agentforce** | AI Agent 自主执行业务 | 按对话/按事务（$2/对话） | $2 per conversation（2025年上线） |
| **微软 Copilot** | Office/业务 AI 助手 | 按用户/月订阅 | $30/用户/月 |

### 2.2 收费模式趋势

2024-2025 年行业出现明显转向：

**从"按坐席/按功能"→"按效果/按事务"**

- **Salesforce Agentforce**（2025）：$2/conversation，按 AI Agent 完成的事务收费，不再按用户数。这是全球 SaaS 巨头首次将 AI Agent 定价从"订阅"转向"按件"。
- **硅基智能**：数字人直播按 GMV 1-5% 提成，是国内最接近"数字员工提成"的先例。
- **百度智能云**：数字人客服按接待量阶梯计费，越用越便宜。

**关键信号**：行业头部已验证"按效果收费"的 AI Agent 定价模型是可行且被客户接受的。

### 2.3 与 Ami "数字员工"模式的对标

| 对标维度 | Salesforce Agentforce | 硅基智能数字人 | Ami_Aura 数字员工 |
|----------|----------------------|--------------|------------------|
| 定价单位 | 每次 AI 对话 $2 | 月费 + GMV 提成 | 工时底薪 + 参与订单提成 |
| 行业 | 通用企业 | 电商直播 | 美业门店 |
| 使用场景 | 客服/销售/HR | 直播卖货 | 接待/核销/推荐/收银 |
| 老板心智 | "AI 帮我处理工单" | "数字人帮我卖货" | "数字员工帮我服务客户" |
| 归因难度 | 低（对话=事务） | 中（GMV 可追踪） | 中高（需区分强/弱参与） |

## 三、按效果付费/提成模式在 SaaS 行业的实践

### 3.1 成功案例

| 公司 | 行业 | 模式 | 效果 |
|------|------|------|------|
| **有赞** | 电商 SaaS | SaaS 年费 + 交易抽成 0.6% | 2023 年交易抽成收入占总营收约 40% |
| **微盟** | 电商/零售 SaaS | SaaS + 精准营销佣金 | 广告+增值服务收入占比超过 50% |
| **Shopify** | 全球电商 | SaaS 月费 + GMV 抽成（Payments 2.9%+30¢） | Merchant Solutions（含支付抽成）收入 > 订阅收入 |
| **Toast**（美国餐饮） | 餐饮 SaaS | 软件费低（免费起步）+ 支付抽成 2.49%+15¢ | 支付及金融收入占总营收 80%+ |
| **Square** | 零售/餐饮 | 硬件低价 + 交易抽成 2.6%+10¢ | 交易收入为绝对主体 |
| **拼多多 Temu** | 电商 | 零入驻费 + 佣金抽成 | 成功激发中小商家参与意愿 |

### 3.2 美业相关的效果付费先例

| 模式 | 案例 | 适用性分析 |
|------|------|-----------|
| **团购佣金** | 美团美业 8-15% | 已被美业老板接受，但怨言大（"给平台打工"） |
| **到店核销分成** | 大众点评推广通 | 按到店转化付费，门店可接受 |
| **会员储值手续费** | 部分 SaaS 对储值金额收 0.5-1% | 金融属性，合规敏感 |
| **营销效果付费** | 有赞精选联盟（按 CPS） | 按实际销售付佣金 |

### 3.3 "提成模式"的关键成功因素

通过上述案例总结：

1. **归因必须清晰可见**：Shopify/Toast 靠支付通道做归因，每笔钱都过系统，不存在争议
2. **基础费要低到门槛感弱**：Toast 的软件近乎免费，靠交易赚钱；老板不觉得"进门就被薅"
3. **提成率必须低于替代成本**：美团 15% 被骂，但 2-3% 的支付手续费无人抱怨
4. **老板能实时看到"替我赚了多少"**：否则只看到扣费，看不到收益

## 四、美业门店对"数字员工"的接受度分析

### 4.1 门店人工成本现状

| 岗位 | 月薪范围（一二线城市） | 核心工作 |
|------|----------------------|----------|
| 前台/接待 | 4000-7000 | 预约登记、接待分流、基础收银 |
| 顾问/销售 | 8000-15000（含提成） | 推荐项目、办卡、客户维护 |
| 美容师 | 6000-12000（含提成） | 服务执行、护理 |
| 店长 | 10000-20000 | 经营管理、报表、团队 |

**关键洞察**：美业门店人工成本占营收 35-50%，老板对"省人"的诉求极强。

### 4.2 接受度判断

**有利因素**：

- 美业老板**已经习惯按"人"算账**，对人效、坪效的概念敏感
- 门店招聘难、流动大，尤其前台岗位，数字化替代有真实需求
- 疫情后美业门店对数字化投入意愿明显提升（2023-2024 多份行业报告证实）
- 美团的佣金模式已经教育了"按效果付费"心智

**阻力因素**：

- **对"被抽成"的本能抵触**：美团佣金的负面印象根深蒂固
- **信任门槛高**：老板需要"看得见"数字员工在干什么、干了多少
- **员工抵触**：现有员工会担心被替代或分薄提成
- **小店付费能力弱**：月营收 10 万以下的小店，3% = 3000/月，可能超过预期

### 4.3 目标客户画像

"数字员工"模式最适合的客户：

| 维度 | 理想客户 | 不适合客户 |
|------|----------|-----------|
| 规模 | 月营收 20-100 万 | 月营收 < 10 万的社区小店 |
| 岗位配置 | 有前台+顾问+美容师分工 | 老板自己兼所有角色 |
| 数字化意愿 | 已用过管理软件 | 纸笔记账、微信沟通 |
| 痛点 | 前台招聘难/流动大、推荐转化低 | 主要是成本控制问题 |
| 客单价 | 500+ | < 100（快剪/基础美甲） |

## 五、竞争定位与差异化建议

### 5.1 市场定位矩阵

```
                    固定订阅收费 ←——————————→ 按效果收费
                         |                        |
     纯管理工具     博卡、美盈易                   |
                         |                        |
     管理+营销      有赞美业          美团美业（佣金）
                         |                        |
     AI 辅助       悦客、艾优你                    |
                         |                        |
     数字员工           —— 空白 ——          ★ Ami_Aura
                         |                        |
```

**Ami_Aura 占据的是"AI 数字员工 × 按效果收费"象限——当前美业市场的绝对空白。**

### 5.2 与竞品的核心差异

| 差异点 | Ami_Aura | 传统美业 SaaS | 美团模式 |
|--------|----------|--------------|----------|
| 产品形态 | 终端设备 + AI Agent | 纯后台软件 | 平台流量入口 |
| 价值感知 | 老板看得见"它在服务客户" | 老板只看后台报表 | 老板看得见订单，但客户是美团的 |
| 定价心智 | "雇了个数字前台" | "买了个管理工具" | "给平台交保护费" |
| 客户归属 | 客户数据属于门店 | 客户数据属于门店 | 客户属于平台 |
| 提成率 | 1.5-3%（远低于人工提成） | 无 | 8-15%（怨声载道） |
| 换用成本 | 高（终端+数据+习惯） | 中（数据迁移） | 低（随时停投） |

### 5.3 定价建议（最终版）

基于竞品分析和市场接受度，建议三档定价：

**青铜版（入门体验）**
- 底薪：1,500/月
- 提成：无
- 适合：想先试用再决定的门店
- 心智："先试一个月，效果好再升级"

**白银版（标准数字员工）**
- 底薪：2,000/月
- 强参与提成：2.5%（终端直接完成的交易）
- 弱参与提成：1%（终端推荐后 24h 内成交）
- 月提成上限：12,000
- 适合：月营收 20-60 万的单店

**黄金版（连锁/高营收门店）**
- 底薪：2,500/月
- 强参与提成：2%（阶梯递减）
- 弱参与提成：0.8%
- 月提成上限：20,000
- 含：多终端部署、专属运营顾问
- 适合：月营收 60 万+ 或 3 家以上连锁

**关键设计原则**：
- 底薪 < 市面半个前台薪资（降低试用门槛）
- 提成率 < 美团佣金的 1/4（避免"抽血"印象）
- 上限封顶（消除大店"越赚越被薅"的恐惧）
- 提供实时可视的"数字员工绩效看板"（建立信任）

## 六、风险与对策

| 风险 | 严重程度 | 对策 |
|------|----------|------|
| 归因争议（"不是它的功劳"） | 高 | 只对"强参与"收全额提成；提供每笔归因明细可查可申诉 |
| 美团佣金类比（"又一个来抽血的"） | 高 | 强调：客户是你的、数据是你的、Ami 只是你的员工不是平台 |
| 员工抵触 | 中 | 将 Ami 定位为"辅助"不是"替代"，强调让员工专注高价值工作 |
| 小店付费能力 | 中 | 提供青铜版（固定低价无提成），低门槛养成习惯 |
| 提成上限导致大店价值感低 | 低 | 超上限后提供增值服务（AI 营销包、连锁管理）作为"加薪"通道 |
| 合规/发票 | 低 | 以"技术服务费"开票，提成部分按"效果推广服务费"处理 |

## 七、结论

1. **模式可行**："工时+提成"在美业 SaaS 是空白定位，全球已有 Salesforce Agentforce（$2/对话）和硅基智能（GMV 提成）验证了 AI Agent 按效果收费的商业逻辑。

2. **时机合适**：2024-2025 年 AI Agent 定价范式转变（从订阅到按事务），美业门店对数字化和省人的需求持续上升。

3. **护城河明确**：终端设备 + 业务数据 + 归因体系构成高换用成本；提成模式让平台和门店利益绑定（门店赚得多→Ami 收得多→有动力优化产品）。

4. **最大风险是信任**：需要"绩效透明化"作为核心产品功能（让老板每天看到数字员工"赚了多少钱"），而非后台静默扣费。

5. **建议先以"白银版"切入 5-10 家种子门店验证**，核心验证：归因是否被接受、提成率是否合理、老板是否续费。
