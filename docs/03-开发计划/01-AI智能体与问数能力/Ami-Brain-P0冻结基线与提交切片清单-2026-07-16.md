### 5.3 兼容策略

完整 `BrainSemanticIntent` 使用 Draft-07 复杂 schema。中转严格 JSON Schema 接口对此返回 HTTP 400，因此运行配置固定为：

1. Terra 输出 JSON Object。
2. 服务端 Ajv 按完整 schema 校验。
3. 校验失败只允许一次脱敏后的受控修复。
4. 仍失败则关闭执行，不进入工具调用。

这不是放松结构约束，而是把最终 schema 执法放回 Ami Core 自己控制。
