## 1. 背景与诊断

### 1.1 当前评测结论

来源：`docs/04-测试数据/Agent评测与知识治理-2026-06-30至07-03/ami-brain-eval-run-2026-07-10/ami-brain-eval-report-2026-07-10.md`

- 题库：`docs/04-测试数据/Agent评测与知识治理-2026-06-30至07-03/agent-eval-questions.md`
- 请求路径：`BrainChatService.createConversation + BrainChatService.sendMessage`
- 总题数：650
- 当前报告可用数：101
- 当前报告可用率：15.5%
- 意图未覆盖：549
- 主要问题：现有评分把“返回 metric citation”直接判为可用，未校验是否答对问题。

### 1.2 真实产品问题

当前 Ami Brain 的实质状态：

1. 主链路已经接通：会话、消息、run、run step 已真实落库。
2. 已支持指标不再返回硬编码 0，`paid_revenue`、`card_liability`、`gross_margin`、`repurchase_rate` 等走真实 SQL。
3. 未支持能力保持拒答，不伪造数据。
4. 关键词触发误命中严重：文案、推荐、操作、排名、名单题会被子串命中成单指标问答。
5. 时间解析残缺：`明天`、`昨天`、`下午`、`现在`、`季度` 等未识别时退化为全量。
6. 指标查询只支持全店单值，不支持分组、排序、名单、对比差值。

### 1.3 修复原则

- 评测优先：先让“可用率”反映真实答题质量，再扩能力。
- 安全优先：宁可拒答，也不返回错误粒度、全量误导值或估算值。
- 意图优先：先判断用户要问数、排名、名单、对比、文案、操作、推荐，再决定是否执行指标。
- 时间必解析：出现时间词但无法解析时，返回澄清或拒答，不退化为全量。
- 粒度必匹配：排名题必须返回排名，名单题必须返回名单，对比题必须返回差值。
- TDD 执行：每个运行时修复先写失败测试，再实现。

## 3. 阶段划分

### Phase P0-A：修评测尺子，让可用率可信

目标：把“可用”从 `completed + citation` 改为 `completed + citation + 意图匹配 + 粒度匹配 + 口径匹配`。

验收：

- 报告中列出的 5 个假阳性样本全部被判为 `false_positive_*`。
- 新报告同时输出旧口径可用率和新口径真实可用率。
- `ami-brain-eval-results-*.json` 每条记录包含 `grader` 字段。

### Phase P0-B：补时间解析，禁止时间词退化全量

目标：`明天/昨天/下午/现在/季度/去年同期` 等时间表达必须解析为明确区间；无法解析时拒答或澄清。

验收：

- “明天预约多少”不返回全量 384。
- “下午还有几个预约”使用今日下午或从当前时刻到今日结束的区间。
- “去年同期收入”进入对比口径或明确拒答，不返回全量。

### Phase P1-A：意图门控优先于指标关键词

目标：文案、动作、推荐、诊断、名单、排名、对比题不再被单个关键词误触发 scalar 指标。

验收：

- “写一条提醒客户预约空档的消息”不执行 `appointment_count`。
- “这次服务完推荐她预约哪个项目”不执行 `appointment_count`。
- “这个月谁的业绩最好”不返回全店 `paid_revenue` 单值。

### Phase P1-B：支持基础分组、排序、对比

目标：对最常见的排名/名单/对比问题提供真实粒度结果；暂未接入的领域继续拒答。

首批支持：

- 收入对比：本月 vs 上月、本周 vs 上周、今天 vs 昨天。
- 预约统计：按日期、项目、美容师分组。
- 员工业绩：按美容师统计实收流水 Top N。
- 复购率：仅回答全店复购率；“谁的客户复购率最高”先拒答，直到员工维度口径实现。
- 会员卡负债：仅回答全店负债；“哪些客户”先拒答，直到名单口径实现。

验收：

- “这个月跟上个月比收入差多少”返回本月、上月、差值、变化率。
- “这个月谁的业绩最好”返回美容师排行榜。
- “哪些产品毛利率最高”在产品毛利口径接入前返回未接入说明，不返回全店毛利率。

### Phase P2：六角色能力接入

目标：从指标 MVP 进入经营智能体。按角色补真实 domain adapter，不靠统一提示语兜底。

优先顺序：

1. 店长经营：经营概览、异常提醒、目标差距、趋势复盘。
2. 前台接待：预约查询、空档查询、预约变更动作预览。
3. 财务风控：收入对账、退款折扣、毛利与成本风险。
4. 库存采购：缺货、临期、消耗、采购建议。
5. 营销增长：客群识别、活动效果、文案生成。
6. 美容师服务：今日安排、服务建议、跟进记录。

创建 `packages/server-v2/src/brain/eval/brain-answer-grader.service.ts`：

```ts
import { Injectable } from '@nestjs/common';

export type BrainEvalIntent =
  | 'metric_query'
  | 'ranking'
  | 'list'
  | 'comparison'
  | 'draft'
  | 'action'
  | 'recommendation'
  | 'diagnosis'
  | 'unknown';

export type BrainEvalAnswerShape =
  | 'scalar_metric'
  | 'ranking'
  | 'list'
  | 'comparison'
  | 'draft'
  | 'action_preview'
  | 'refusal'
  | 'empty';

export type BrainEvalGradeStatus =
  | 'usable_exact'
  | 'false_positive_intent_mismatch'
  | 'false_positive_granularity_mismatch'
  | 'correct_refusal'
  | 'metric_failed'
  | 'error';

export interface BrainAnswerGradeInput {
  question: string;
  answer: string;
  citations: Array<{ sourceType?: string; sourceId?: string; label?: string }>;
  brainStatus?: string;
}

export interface BrainAnswerGradeResult {
  status: BrainEvalGradeStatus;
  expectedIntent: BrainEvalIntent;
  actualIntent: BrainEvalIntent;
  expectedShape: BrainEvalAnswerShape;
  actualShape: BrainEvalAnswerShape;
  reason: string;
}

@Injectable()
export class BrainAnswerGraderService {
  grade(input: BrainAnswerGradeInput): BrainAnswerGradeResult {
    const expectedIntent = this.detectExpectedIntent(input.question);
    const expectedShape = this.expectedShapeForIntent(expectedIntent);
    const actualShape = this.detectActualShape(input.answer, input.citations);
    const actualIntent = actualShape === 'scalar_metric' ? 'metric_query' : expectedIntent;

    if (input.brainStatus === 'failed') {
      return { status: 'metric_failed', expectedIntent, actualIntent, expectedShape, actualShape, reason: 'brain_status_failed' };
    }

    if (actualShape === 'refusal') {
      return { status: 'correct_refusal', expectedIntent, actualIntent, expectedShape, actualShape, reason: 'explicit_refusal' };
    }

    if (expectedIntent !== 'metric_query' && expectedIntent !== actualIntent) {
      return {
        status: 'false_positive_intent_mismatch',
        expectedIntent,
        actualIntent,
        expectedShape,
        actualShape,
        reason: `expected_${expectedIntent}_but_answered_${actualIntent}`,
      };
    }

    if (expectedShape !== actualShape) {
      return {
        status: 'false_positive_granularity_mismatch',
        expectedIntent,
        actualIntent,
        expectedShape,
        actualShape,
        reason: `expected_${expectedShape}_but_answered_${actualShape}`,
      };
    }

    return { status: 'usable_exact', expectedIntent, actualIntent, expectedShape, actualShape, reason: 'intent_and_shape_matched' };
  }

  private detectExpectedIntent(question: string): BrainEvalIntent {
    if (/(写|生成|编辑|拟|话术|短信|文案|消息|朋友圈|推送)/.test(question)) return 'draft';
    if (/(创建|新增|安排|取消|改约|预约|下单|采购|核销|退款|结算)/.test(question) && /(帮我|给|把|处理|操作)/.test(question)) return 'action';
    if (/(推荐|建议).*(项目|产品|方案|护理|活动)/.test(question)) return 'recommendation';
    if (/(为什么|原因|诊断|分析|异常|风险)/.test(question)) return 'diagnosis';
    if (/(比|相比|差多少|环比|同比|增长|下降|趋势)/.test(question)) return 'comparison';
    if (/(谁|哪个|哪些|排行|排名|最高|最多|最好|最低|最少|Top|TOP)/.test(question)) {
      if (/(哪些|名单|列一下|找一下)/.test(question)) return 'list';
      return 'ranking';
    }
    if (/(多少|几个|几笔|金额|流水|收入|毛利|复购率|预约数|缺货|负债)/.test(question)) return 'metric_query';
    return 'unknown';
  }

  private expectedShapeForIntent(intent: BrainEvalIntent): BrainEvalAnswerShape {
    if (intent === 'ranking') return 'ranking';
    if (intent === 'list') return 'list';
    if (intent === 'comparison') return 'comparison';
    if (intent === 'draft') return 'draft';
    if (intent === 'action') return 'action_preview';
    if (intent === 'metric_query') return 'scalar_metric';
    return 'refusal';
  }

  private detectActualShape(answer: string, citations: BrainAnswerGradeInput['citations']): BrainEvalAnswerShape {
    if (!answer) return 'empty';
    if (/(尚未|不能|无法|不会编造|未接入|请提问已注册指标|能力边界)/.test(answer)) return 'refusal';
    if (/(第1|第一|Top|TOP|排名|排行榜)/.test(answer)) return 'ranking';
    if (/(名单|如下|1\.|2\.|客户：|项目：)/.test(answer)) return 'list';
    if (/(上月|上周|昨天|同期|差值|变化率|增长|下降)/.test(answer)) return 'comparison';
```

```ts
import { Injectable } from '@nestjs/common';

export type BrainTimeGranularity =
  | 'day'
  | 'week'
  | 'month'
  | 'quarter'
  | 'year'
  | 'time_window'
  | 'remaining_today'
  | 'none';

export interface BrainParsedTimeRange {
  matched: boolean;
  hasTemporalIntent: boolean;
  source?: string;
  granularity: BrainTimeGranularity;
  startDate?: Date;
  endDate?: Date;
}

@Injectable()
export class BrainTimeRangeParserService {
  parse(text: string, now = new Date()): BrainParsedTimeRange {
    const localNow = this.toChinaLocal(now);

    if (/现在|当前|此刻|接下来|还有/.test(text)) {
      return this.range('现在', 'remaining_today', now, this.endOfChinaDay(localNow));
    }
    if (/明天/.test(text)) return this.dayRange('明天', this.addDays(localNow, 1));
    if (/昨天/.test(text)) return this.dayRange('昨天', this.addDays(localNow, -1));
    if (/今天|今日/.test(text)) return this.dayRange('今天', localNow);
    if (/上午/.test(text)) return this.range('上午', 'time_window', this.atChinaTime(localNow, 0, 0, 0, 0), this.atChinaTime(localNow, 11, 59, 59, 999));
    if (/下午/.test(text)) return this.range('下午', 'time_window', this.atChinaTime(localNow, 12, 0, 0, 0), this.atChinaTime(localNow, 17, 59, 59, 999));
    if (/晚上|今晚/.test(text)) return this.range('晚上', 'time_window', this.atChinaTime(localNow, 18, 0, 0, 0), this.atChinaTime(localNow, 23, 59, 59, 999));
    if (/本周|这周/.test(text)) return this.weekRange('本周', localNow, 0);
    if (/上周/.test(text)) return this.weekRange('上周', localNow, -1);
    if (/本月|这个月/.test(text)) return this.monthRange('本月', localNow, 0);
    if (/上月|上个月/.test(text)) return this.monthRange('上月', localNow, -1);
    if (/本季度|这个季度/.test(text)) return this.quarterRange('本季度', localNow, 0);
    if (/上季度|上个季度/.test(text)) return this.quarterRange('上季度', localNow, -1);

    const unresolved = text.match(/去年同期|同比|同期|季度|明年|去年|今年/);
    if (unresolved) return { matched: false, hasTemporalIntent: true, source: unresolved[0], granularity: 'none' };

    return { matched: false, hasTemporalIntent: false, granularity: 'none' };
  }

  private dayRange(source: string, localDate: Date): BrainParsedTimeRange {
    return this.range(source, 'day', this.atChinaTime(localDate, 0, 0, 0, 0), this.atChinaTime(localDate, 23, 59, 59, 999));
  }

  private weekRange(source: string, localDate: Date, offsetWeeks: number): BrainParsedTimeRange {
    const day = localDate.getDay() || 7;
    const monday = this.addDays(localDate, 1 - day + offsetWeeks * 7);
    const sunday = this.addDays(monday, 6);
    return this.range(source, 'week', this.atChinaTime(monday, 0, 0, 0, 0), this.atChinaTime(sunday, 23, 59, 59, 999));
  }

  private monthRange(source: string, localDate: Date, offsetMonths: number): BrainParsedTimeRange {
    const start = new Date(localDate.getFullYear(), localDate.getMonth() + offsetMonths, 1);
    const end = new Date(localDate.getFullYear(), localDate.getMonth() + offsetMonths + 1, 0);
    return this.range(source, 'month', this.atChinaTime(start, 0, 0, 0, 0), this.atChinaTime(end, 23, 59, 59, 999));
  }

  private quarterRange(source: string, localDate: Date, offsetQuarters: number): BrainParsedTimeRange {
    const quarterStartMonth = Math.floor(localDate.getMonth() / 3) * 3 + offsetQuarters * 3;
    const start = new Date(localDate.getFullYear(), quarterStartMonth, 1);
    const end = new Date(localDate.getFullYear(), quarterStartMonth + 3, 0);
    return this.range(source, 'quarter', this.atChinaTime(start, 0, 0, 0, 0), this.atChinaTime(end, 23, 59, 59, 999));
  }

  private range(source: string, granularity: BrainTimeGranularity, startDate: Date, endDate: Date): BrainParsedTimeRange {
    return { matched: true, hasTemporalIntent: true, source, granularity, startDate, endDate };
  }

  private toChinaLocal(date: Date) {
    return new Date(date.getTime() + 8 * 60 * 60 * 1000);
  }

  private atChinaTime(localDate: Date, hours: number, minutes: number, seconds: number, milliseconds: number) {
    return new Date(Date.UTC(localDate.getFullYear(), localDate.getMonth(), localDate.getDate(), hours - 8, minutes, seconds, milliseconds));
  }

  private endOfChinaDay(localDate: Date) {
    return this.atChinaTime(localDate, 23, 59, 59, 999);
  }

  private addDays(date: Date, days: number) {
    const copy = new Date(date);
    copy.setDate(copy.getDate() + days);
    return copy;
  }
}
```

```ts
import { Injectable } from '@nestjs/common';

export interface BrainAnswerComposerInput {
  shape: 'scalar' | 'comparison' | 'ranking' | 'list';
  label: string;
  metric: string;
  valueField?: string;
  rows: Array<Record<string, unknown>>;
}

@Injectable()
export class BrainAnswerComposerService {
  compose(input: BrainAnswerComposerInput) {
    if (input.shape === 'comparison') return this.composeComparison(input);
    if (input.shape === 'ranking') return this.composeRanking(input);
    if (input.shape === 'list') return this.composeList(input);
    return this.composeScalar(input);
  }

  private composeScalar(input: BrainAnswerComposerInput) {
    const value = this.numberValue(input.rows[0]?.[input.valueField ?? input.metric]);
    return `${input.label}为 ${this.formatValue(input.metric, value)}。`;
  }

  private composeComparison(input: BrainAnswerComposerInput) {
    const row = input.rows[0] ?? {};
    const current = this.numberValue(row.current_value);
    const previous = this.numberValue(row.previous_value);
    const delta = this.numberValue(row.delta_value);
    const rate = this.numberValue(row.delta_rate);
    return `${input.label}本期为 ${this.formatValue(input.metric, current)}，上期为 ${this.formatValue(input.metric, previous)}，差值为 ${this.formatValue(input.metric, delta)}，变化率为 ${(rate * 100).toFixed(1)}%。`;
  }

  private composeRanking(input: BrainAnswerComposerInput) {
    if (!input.rows[0]?.rank_label) {
      return '当前查询需要排名结果，但 Ami Brain 只拿到了单值指标，系统已停止返回不匹配答案。';
    }
    return input.rows
      .slice(0, 5)
      .map((row, index) => `${index + 1}. ${String(row.rank_label)}：${this.formatValue(input.metric, this.numberValue(row.metric_value))}`)
      .join('\n');
  }

  private composeList(input: BrainAnswerComposerInput) {
    if (!input.rows[0]?.item_label) {
      return '当前查询需要名单结果，但 Ami Brain 只拿到了单值指标，系统已停止返回不匹配答案。';
    }
    return input.rows.slice(0, 10).map((row) => `- ${String(row.item_label)}`).join('\n');
  }

  private numberValue(value: unknown) {
    if (typeof value === 'number') return value;
    if (typeof value === 'bigint') return Number(value);
    if (typeof value === 'string') return Number(value);
    if (value && typeof value === 'object' && 'toString' in value) return Number(value.toString());
    return 0;
  }

  private formatValue(metric: string, value: number) {
    if (metric.endsWith('_rate') || metric === 'repurchase_rate') return `${(value * 100).toFixed(1)}%`;
    if (metric.includes('revenue') || metric.includes('margin') || metric.includes('liability') || metric.includes('value')) return `${value.toFixed(2)} 元`;
    return String(Math.round(value));
  }
}
```
