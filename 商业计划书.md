# 美业管理平台 — 商业计划书

## 一、执行摘要

美业管理平台是一款面向美容院、美容养生会所及美业连锁品牌的 SaaS 数字化经营管理系统。平台以「智能营销 + 供应链管理」为核心差异化能力，覆盖客户管理、营销自动化、门店运营、商品订单、库存供应链及系统管理 8 大模块 33 个功能页面，帮助美业经营者降低运营成本、提升客户复购率和门店坪效。

中国美业市场规模超 6000 亿元，但数字化渗透率不足 15%。大量中小美容院仍依赖纸质记录和 Excel 管理，面临客户流失率高、库存损耗大、营销效率低等痛点。本产品瞄准这一市场空白，以低门槛 SaaS 订阅模式切入，目标在 3 年内服务 5000+ 门店。

## 二、市场分析

### 2.1 行业概况

| 指标 | 数据 |
|------|------|
| 中国美业市场规模（2025年） | 约 6500 亿元 |
| 年均增长率 | 8-12% |
| 美容院/会所数量 | 约 400 万家 |
| 连锁品牌占比 | 约 15%，且快速增长 |
| 数字化渗透率 | 不足 15% |

### 2.2 目标市场

按门店规模分层定位：

| 层级 | 门店特征 | 规模 | 切入策略 |
|------|----------|------|----------|
| 核心市场 | 3-20 家门店的连锁品牌 | 约 8 万家 | 主力目标，全功能套餐 |
| 扩展市场 | 单店美容院/会所 | 约 350 万家 | 轻量版切入，逐步升级 |
| 高端市场 | 20 家以上大型连锁 | 约 2 万家 | 定制化方案 |

### 2.3 竞争格局

| 竞品类型 | 代表产品 | 优势 | 劣势 |
|----------|----------|------|------|
| 通用 SaaS | 有赞、微盟 | 品牌知名度高 | 美业场景适配差，无库存/BOM |
| 美业垂直 SaaS | 博卡、美管加 | 行业经验丰富 | 产品老旧，AI 能力弱 |
| ERP 系统 | 用友、金蝶 | 供应链能力强 | 价格高，部署重，不适合中小门店 |

### 2.4 竞争优势

1. AI 智能营销：自动化策略引擎（10 种触发规则 × 6 种执行动作），竞品多为手动配置
2. 供应链闭环：从 BOM 配方 → 消耗追踪 → AI 补货预测 → 采购 → 调拨的完整链路
3. 多门店协同：库存对比矩阵、智能调拨建议、集中/独立采购模式
4. 低使用门槛：现代化 UI，开箱即用，无需培训
5. 灵活定价：按门店数订阅，中小商户可负担

## 三、产品规划

### 3.1 产品矩阵

| 版本 | 目标客户 | 核心功能 | 定价 |
|------|----------|----------|------|
| 基础版 | 单店 | 客户管理 + 订单 + 排班预约 | ¥299/月 |
| 专业版 | 3-10 家连锁 | 基础版 + 智能营销 + 库存管理 | ¥699/月/店 |
| 旗舰版 | 10 家以上连锁 | 专业版 + AI 全功能 + 多门店调拨 + API 开放 | ¥999/月/店 |
| 定制版 | 大型连锁 | 旗舰版 + 私有化部署 + 定制开发 | 按需报价 |

### 3.2 产品路线图

| 阶段 | 时间 | 里程碑 |
|------|------|--------|
| MVP | 2026 Q2 | 后端 API 上线，核心流程跑通（客户 + 订单 + 库存） |
| V1.0 | 2026 Q3 | 全模块上线，首批 50 家门店内测 |
| V1.5 | 2026 Q4 | AI 功能落地（补货预测、营销推荐），小程序端上线 |
| V2.0 | 2027 Q1 | 多门店协同、数据大屏、开放 API |
| V2.5 | 2027 Q2 | 移动端 App、供应商协同门户 |

### 3.3 技术壁垒

- AI 补货预测模型：基于历史销售数据的时间序列预测，考虑季节性和促销因素
- 智能营销引擎：RFM 客户分群 + 规则引擎 + 效果反馈闭环
- BOM 消耗追踪：服务项目与产品消耗的精确关联，偏差异常检测

| 收入来源 | 模式 | 占比预期 |

## 十、总结

美业管理平台抓住了中国美业数字化转型的窗口期，以「AI 智能营销 + 供应链闭环」为核心竞争力，通过 SaaS 订阅模式服务中小美业连锁品牌。产品已完成 33 个功能页面的前端开发，工程化基础就绪，具备快速推向市场的条件。

核心优势：
- 产品完整度高，覆盖美业经营全链路
- AI 能力形成差异化壁垒
- SaaS 模式可规模化复制
- 美业垂直赛道竞争格局未定，存在弯道超车机会

我们正在寻求天使轮融资 300 万元，用于完成后端开发、获取首批种子客户，并在 2027 年实现盈亏平衡。
